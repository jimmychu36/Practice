# 🚀 GitHub Pages 部署完整指南

## 📋 準備工作
✅ 已準備好 `github_pages_index.html` 檔案
✅ 需要 GitHub 帳號（免費申請：github.com）

---

## 🎯 第一步：創建 GitHub 倉庫

### 1. 登入 GitHub
- 到 [github.com](https://github.com) 登入帳號
- 如果沒有帳號，點 "Sign up" 免費註冊

### 2. 創建新倉庫
- 點右上角 "+" → "New repository"
- **Repository name** 填入：`taiwan-elementary-quiz`（可自訂）
- **Description** 填入：`台灣國小練習 App - 540道精選題目`
- 勾選 "Public"（免費用戶只能用公開倉庫）
- 勾選 "Add a README file"
- 點 "Create repository"

---

## 📁 第二步：上傳檔案

### 1. 進入倉庫後點 "uploading an existing file"
### 2. 上傳檔案
- 把 `github_pages_index.html` 拖到上傳區域
- **重要**：把檔名改成 `index.html`（GitHub Pages 的預設首頁）

### 3. 提交變更
- 在下方 "Commit changes" 填入：`Add elementary practice app`
- 點 "Commit changes"

---

## ⚙️ 第三步：啟用 GitHub Pages

### 1. 進入 Settings
- 在倉庫頁面點上方的 "Settings" 標籤

### 2. 找到 Pages 設定
- 在左側選單找到 "Pages"

### 3. 設定 Source
- **Source** 選擇：`Deploy from a branch`
- **Branch** 選擇：`main`
- **Folder** 選擇：`/ (root)`
- 點 "Save"

### 4. 等待部署
- 會出現綠色訊息：`Your site is published at https://你的用戶名.github.io/taiwan-elementary-quiz`
- 部署需要 1-10 分鐘

---

## 🌐 第四步：取得網址

### 你的 App 網址會是：
```
https://你的GitHub用戶名.github.io/倉庫名稱
```

**範例**：
- 如果你的用戶名是 `johnsmith`
- 倉庫名是 `taiwan-elementary-quiz`  
- 網址就是：`https://johnsmith.github.io/taiwan-elementary-quiz`

## 🔧 常見問題解決

### Q: 網址顯示 404？
**A**: 等待 5-10 分鐘，GitHub Pages 需要時間部署

### Q: 檔案沒有音效？
**A**: 確認檔名是 `index.html`，不是其他名稱

### Q: 想要自訂網域？
**A**: 在 Pages 設定中的 "Custom domain" 填入你的網域

### Q: 想更新 App？
**A**: 直接上傳新的 `index.html` 檔案覆蓋即可
